# nvim

Configuration for neovim, using lazy.nvim as plugin manager. Place this folder in 'C:\Users\%USERNAME%\AppData\Local\nvim\' on windows. The init.lua file is the main configuration file for neovim, and it sets up the plugins and the color scheme. The plugins are organized into separate files in the 'lua\plugins\' folder, and the color scheme is set up in the 'lua\colorscheme\' folder. I use the Shades of Purple color scheme. You can find more information about configuring neovim in the [official documentation](https://neovim.io/doc/).
