#Flowlauncher

Flow Launcher is an application that allows you to quickly launch applications, search for files, and perform various tasks using a simple keyboard interface. It is similar to the built-in Windows Start menu, but with more features and customization options. I use Flow Launcher to quickly access my most frequently used applications and files, as well as to perform quick calculations and search the web. It is a great tool for improving productivity and saving time. You can find more information about Flow Launcher and download it from the [official website](https://flow-launcher.com/).

Again, with the Shades of Purple color scheme, I tried to match the color scheme as closely as possible in Flow Launcher. To install the theme, copy the file and place it in the following folder on windows: 'C:\Users\%USERNAME%\AppData\Roaming\FlowLauncher\Themes\'.
