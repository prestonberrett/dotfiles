#Yazi

Yazi is a terminal file manager that I use to navigate my file system. Here is the [link](https://github.com/sxyazi/yazi) to the github page. I have this whole folder in 'C:\Users\%USERNAME%\.config\yazi\', and then I have an alias in my powershell profile to open yazi by just typing 'y' in the terminal. Yazi uses the Catppuccin Frappe color scheme, found [here](https://github.com/catppuccin/yazi). To install the theme, copy the file and place it in the following folder on windows: 'C:\Users\%USERNAME%\.config\yazi\themes\'.
