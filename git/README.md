#git

.gitcommittemplate file contains my default template for git commit messages. It is located in C:\Users\Username\, and it is used when you run 'git commit' without the '-m' flag. It contains a template for the commit message, and it also contains some comments to help me remember what to include in the commit message. You can customize this file to fit your needs, but I find that having a template helps me write better commit messages.

.gitconfig file contains my default global git configuration. It is located in C:\Users\Username\, and it is used when you run 'git config' commands. It contains some default settings for git, such as the default editor, the default merge tool, and some aliases for common git commands.
