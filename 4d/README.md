#README for 4D

The Macros.xml file just contains the default macros for 4D. I have added a couple for comments for when creating a new method, a new 'section', a new 'header', code modifications. There is also a macro for changing to var syntax.
The Macros.xml file needs to be placed in the following folder on windows: 'C:\Users\%USERNAME%\AppData\Roaming\4D\Macros v2\'. 

I really enjoy the [Shades of Purple](https://github.com/ahmadawais/shades-of-purple-vscode) vs code color scheme, the [catppuccin machiatto](https://github.com/catppuccin/catppuccin) color scheme, and [solarized dark](https://ethanschoonover.com/solarized/) color scheme. So, I tried to match those color schemes as closely as possible in 4D. To install a theme, copy the file and place it in the following folder on windows: 'C:\Users\%USERNAME%\AppData\Roaming\4D\4D Editor Themes\'.
