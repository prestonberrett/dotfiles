#Wezterm

Config file for Wezterm terminal emulator. This file is located at `~/.config/wezterm/wezterm.lua`. You can customize the appearance and behavior of Wezterm by modifying this file. Mine sets up the default font to be Zed Mono, the color scheme to be Shades of Purple, and the default cursor style to be a blinking bar. It also sets up some key bindings for opening a new tab, splitting the pane vertically, and splitting the pane horizontally. You can find more information about configuring Wezterm in the [official documentation](https://wezfurlong.org/wezterm/config/files.html).
