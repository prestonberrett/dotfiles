#Starship

Configuration file for the starship prompt. This file is located at `~/.config/starship.toml`. You can customize the appearance and behavior of the starship prompt by modifying this file. Mine sets up the color scheme to be Shades of Purple, and it also sets up some custom modules for displaying the current git branch, the current node version, and the current python version. You can find more information about configuring starship in the [official documentation](https://starship.rs/config/).
