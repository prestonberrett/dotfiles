#GlazeWM
GlazeWM is a window tiling manager for Windows.
[glzr-io/glazewm: GlazeWM is a tiling window manager for Windows inspired by i3wm.](https://github.com/glzr-io/glazewm) is the link to the repository.

When you install GlazeWM it will add a .glzr folder to your user folder (ie. C:\Users\UserName\.glzr)
